项目概述

本项目旨在探索AI工具如何成为数据分析师的"智能副驾"，通过实际案例展示人机协同在数据分析全流程中的应用。

核心价值：
效率提升：将重复性工作自动化，释放分析师创造力
质量保障：人机双重校验，降低人为错误率
可复现性：标准化指令模板，建立可复用工作流
技能转型：从"操作者"到"架构师+质检师"的角色升级
